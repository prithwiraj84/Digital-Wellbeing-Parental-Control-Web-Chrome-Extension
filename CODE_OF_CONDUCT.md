# Code of Conduct

This project follows a standard contributor code of conduct to keep the community welcoming and respectful.

Be respectful
------------
- Treat others with empathy and respect.
- Be open to feedback and constructive criticism.

Unacceptable behavior
---------------------
- Harassment, hateful language, or personal attacks are not tolerated.

Reporting
---------
If you experience or witness unacceptable behavior, please open an issue or contact the project maintainers so we can address it.

Thanks for helping maintain a positive community.
